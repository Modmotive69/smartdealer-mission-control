## Product Portfolio

### 1. DMSbackups (dmsbackups.com)
- **What it does:** Daily backup of Dealer Management System (DMS) data and documents to local secure storage
- **Core tech:** B.A.R.T. (Bidirectional Access Real-Time) — syncs, searches, views, and shares dealership data
- **Key pain point it solves:** Cyberattacks on DMS providers (CDK, Reynolds & Reynolds, etc.) leaving dealers exposed
- **Supported:** 54 CDK file types, 67 Reynolds file types, 10+ DMS/CRM providers
- **Departments covered:** Accounting, Payroll, F&I, Sales, Vehicle Management, CRM, Service, Parts
- **Pricing:**
  - Setup/initial download: $5,000 one-time
  - Backups only: $400/mo
  - Backup + DocEvolution bundle: $799/mo
  - "Advanced" (DealerBox + MyDealerKPI + Backups): $1,999/mo

### 2. DocEvolution (docevolution.com)
- **What it does:** Cloud-based document management system — scan, capture, find, share, sign, automate
- **Key features:** Cloud storage (GCP/AWS/Azure), OCR, digital signatures, no-code workflows, white-label capable
- **Target:** Dealerships going paperless, managing DMS docs, remote teams
- **White label:** Yes — can be offered under partner branding
- **Notable:** Bundled with DMSbackups at $799/mo

### 3. InDesign Data / DMSconnect (indesigndata.com)
- **What it does:** Bidirectional DMS data integration — Push & Pull pipeline powered by B.A.R.T.
- **Target:** Software companies (ISVs) that need DMS data access for their apps
- **Supports:** CDK, Reynolds & Reynolds, and other major DMS providers
- **Key value:** Plug into Sales, Service, Parts, Accounting, CRM data in real-time
- **Customer profile:** B2B — other software vendors, not directly dealerships
